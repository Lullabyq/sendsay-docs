# Как экспортировать базу с данными

Чтобы повторить действия из видеоурока, перейдите в интерфейс [по ссылке](https://sendsay.ru/account/#dashboard), либо откройте меню аккаунта в правом верхнем углу и выберите пункт **Предыдущий интерфейс**.
